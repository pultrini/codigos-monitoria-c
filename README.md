# Monitoria PC 2026 FM

## Objetivo
Disponibilizar os códigos em C e posteriormente em python/linguagem que for de interesse dos alunos.

## Informações Importantes
Os códigos desenvolvidos pelos monitores não necessáriamente são os melhores códigos desenvolvidos, podendo serem melhorados pelos alunos, mas sendo necessário sempre fazer o teste manual
Ex.: Um código de média deve ter o mesmo resultado tanto com as contas feitas na mão, mas como também no algoritmo.

### Contribuições
Os alunos que se confortaveis podem abrir PR com novos arquivos para deixar salvo seu conhecimento salvo dentro deste github. ☺️
